⚡ Node.js Delivery Daemon

The daemon is a REQUIRED background service. It listens to the database and executes real-time RCON commands to drop items into player inventories. Use the INIT SCHEMA button above before running it for the first time.

# 1. Enter the Addon Daemon directory cd runtime/addons/installed/dune-airdrop-addon/daemon/ # 2. Build the Docker image docker build -t airdrop-daemon . # 3. Start the background container (Note: You may need --group-add to allow docker socket access) docker rm -f airdrop-daemon docker run -d \ --name airdrop-daemon \ --network host \ --restart unless-stopped \ -e DUNE_DOCKER_ROOT=/repo \ -v $(pwd)/../../../../../runtime/scripts:/repo/runtime/scripts:ro \ -v $(pwd)/../../../../../.env:/repo/.env:ro \ -v /var/run/docker.sock:/var/run/docker.sock:ro \ airdrop-daemon
